What is the difference between local storage and session storage?

The difference between localStorage and sessionStorage is that the data in localStorage stays the same which means it doesn't expire when the page loads. On the other hand the data in sessionStorage is instantly cleared when the page session ends.

What is the difference between Global scope and block scope?

Global scope are those whose variables are defined outside the function and curly brackets. Global scope variables can also be accessed from different parts of any program. Block scope on the other hand is a sub-type of local scope which means its variables can be defined inside curly brackets, which can be loops and conditionals.

How does Javascript event loop work?

An event loop works by pulling stuff out of a queue and then placing it onto a function execution stack whenever the function stack becomes empty. It works in a cyclic manner, where it continually checks whether or not the call stack is empty. It is single-threaded not multithreaded.

What are the many ways we can get undefined

Undefined is a primitive value in JavaScript. A variable has an undefined value when no value is assigned before using it. For example:

  • When a value is not assigned to a variable
  • When less arguments is passed in a function, it results in the parameter having a undefined value
  • When a variable is not declared at all
  • When we try to access non-existing object properties
  • A never returning function can result in undefined
  • Accessing array elements outside of the index range
  • If return has nothing on the right side will give you undefined
  • Setting a value directly to undefined
